# BatchImageCropper
Toplu fotoğraf kırpma uygulaması.
Çoklu fotoğraflarda bağımsız olarak seçim alanı oluşturup kırpma işlemini uygulana bilir
Birden fazla fotoğraf atıp senkron seçeneğini açarsanız çalışma yaptığınız fotoğraftaki kırpma alanı açık olan tüm fotoğraflara uygulanır, senkronu kapatığınızda yine bağımsız olarak düzenelemeye devam edebilirsiniz. 
Meta bilgisi koru aktif olduğunda bilgiler ve zaman damgaları kaynak fotoğrafla aynı olur.
Kırpılan alanlar kaynak dosya ile aynı alanda oluşturulur.
